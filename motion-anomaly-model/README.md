# SafeRoute+

A continuous safety and wellness companion for Nigerian women.
Built for the HER Hackathon by NITHUB (University of Lagos), track: Women's Safety & Health.

## Getting started

1. Copy `.env.example` to `.env` and fill in any required values.
2. Each module folder has its own README with run instructions.
3. Start with `backend-api` — most modules depend on it.
