# design

Mockups, design tokens, component styling reference.

**Owner:** UI/UX Designer.
**Status:** placeholder — the draft "Quiet Vigilance" direction lives in [AGENT.md](../AGENT.md). The owner's own mockups, once shared, take precedence over the placeholder theme.

## What goes here

- High-fidelity mockups (Figma exports, PNGs)
- Design tokens (colors, spacing, type scale) in a portable format
- Component styling reference (states for idle vs. alert, the "safety pulse" animation spec)

## Draft direction (subject to designer override)

See AGENT.md → "Design Direction — 'Quiet Vigilance'" for the placeholder palette and typography. Once the designer ships her own direction, **update both this folder and AGENT.md** to reflect hers.

## Demo Day talking point (for the owner)

"We avoided the loud-red emergency-app cliché. The app is calm by default — the visual urgency is held in reserve and only released when something is actually wrong. That contrast is the experience."
